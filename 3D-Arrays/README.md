# Multi-Dimensional Data Analysis - The Sales Cube

## Overview
A data management application designed to process and analyze automotive sales data across three dimensions: Time (Days), Brands, and Vehicle Types.

## Key Features
* **3D Array Implementation:** Uses a `cube[DAYS][BRANDS][TYPES]` structure to manage large datasets efficiently.
* **Delta Metrics:** Calculates the average rate of change in sales across consecutive days for specific brands.
* **Aggregated Insights:** Identifies top-performing brands, vehicle types, and most profitable days across the entire dataset.
* **Data Validation:** Robust handling of incomplete daily data and brand-specific updates.

## Skills Demonstrated
* Multi-dimensional Array Manipulation
* Data Aggregation and Statistical Reporting
* Floating-point precision arithmetic
